What is Meta Data?

Meta Data can be regaurded into three different category

Category One

Comments

One trick that many coders using html is comments. Comments are tags that can only be seen in the WYSIWYG. This is useful for creating small notes to remember later, naming when certain websites or images are accessed, or for comminication between coders working on the same website. They look like this:

<!--This is a comment! (although you cant exactly see a real comment since this is just example text)-->


Category Two

Browser Search Tags

Meta Data tags are used in the <'head> tag, an example would be:

<html>
<head>
<meta name="author" content="*your name inserted here*">
<meta name="description" content="*say what your website is about">
</head>
</html>

These tags help people find your website. When someone searches for the words in the "content" attribute, google will display your website.

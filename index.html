When making a website, you can make the code easier to understand/visualize by creating different tags corresponding to the placement of the text/code you want it in.
Example: the use of "header", "nav", "aside", "main", "section", "article", "footer"
By stylizing the use of these tags you are able to create a more pleasing looking code that isnt spaghetti code.


Once creating an organized code that is to your liking you can now begin the process of adding attributes to the tags
Example: the use of <'a href="*hyperlink*" target="_blank">
The href is a attribute that links another website, webpage, or file. I honestly dont exactly remember why the "_blank" is needed so think of it what you will
